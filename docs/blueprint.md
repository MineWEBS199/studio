# **App Name**: Trivia Master

## Core Features:

- Question Generation: Generate a trivia question and its possible answers based on general knowledge topics.
- Answer Selection: Allow the user to select one of the possible answers.
- Score Calculation: Keep track of the user's score and update it based on the correctness of their answers.
- Difficulty Level Adjustment: Adjust the difficulty of the trivia questions.
- AI Assistance Tool: Provide hints using AI to guide users toward the correct answer using reasoning to decide if and when to expose it.

## Style Guidelines:

- Primary color: Royal blue (#4169E1) for a scholarly and trustworthy feel.
- Background color: Light gray (#D3D3D3) to ensure comfortable readability.
- Accent color: Yellow-gold (#FFD700) to highlight correct answers and important UI elements.
- Body and headline font: 'PT Sans', a humanist sans-serif providing a modern, yet slightly warm feel, will be used.
- Use clear and intuitive icons to represent different categories and actions within the trivia app.
- Use a clean and organized layout with clear visual hierarchy to guide users through the trivia questions and answers.
- Subtle animations on question transitions to enhance the user experience.
